# 🛡️ 우리말 수호대 - 맞춤법 배틀 게임 (최종판)

초등학생들이 가장 자주 틀리고 헷갈려 하는 맞춤법을 전자칠판이나 태블릿에서 직접 터치하며 재미있게 익힐 수 있는 인터랙티브 웹 게임입니다. 단일 HTML 파일로 제작되어 별도의 서버 없이 즉시 실행 가능합니다.

## ✨ 주요 특징 및 기술 스택
- **기술 스택**: React, Babel(Standalone), Tailwind CSS, Animate.css
- **디자인 컨셉**: 최신 트렌드인 **뉴모피즘(Neumorphism)** 스타일을 적용하여 부드럽고 입체적인 UI 구성. 질문 박스에 오렌지/살구색 그라데이션을 주어 직관적인 시인성 확보.
- **반응형 & 최적화**: 스마트폰부터 대형 전자칠판까지 하단 잘림(Clipping) 없이 모든 기기에 완벽히 맞도록 레이아웃 최적화. 터치 딜레이 및 텍스트 드래그 방지 처리.
- **플로팅 UI**: 홈 버튼과 다크 모드 토글 버튼을 하단 구석에 플로팅 형태로 배치하여 쾌적한 화면 영역 확보.

## 🎮 주요 기능 소개

### 1. 다양한 게임 모드 설정
- **1인 학습 모드**: 혼자서 차분하게 맞춤법을 공부하고 복습할 수 있는 모드.
- **2인 대전 모드**: 화면이 반으로 나뉘어 두 친구가 동시에 대결하는 스피드 경쟁 모드.
- **문항 수 선택**: 수업 시간이나 남는 시간에 맞춰 10문제 / 20문제 / 30문제 중 자유롭게 선택 가능.

### 2. 다이내믹한 2인 대전 모드 (핵심 기능)
- **콜드 게임 & 줄다리기**: 상단의 줄다리기 게이지 바를 통해 점수 차이를 시각적으로 확인. 한 팀이 5점 차이로 앞서나가면 즉시 '콜드 게임 승리'로 종료되어 긴장감 극대화.
- **즉각적인 정답 전환**: 정답을 맞히는 순간 즉시 다음 문제로 넘어가 박진감 넘치는 템포 제공.
- **학습 효과 극대화 (팝업 피드백)**: 한 플레이어가 오답을 누르면, 양쪽 플레이어 모두의 화면 중앙에 **강렬한 팝업창으로 올바른 정답과 해설을 즉시 띄워줍니다.** 단순 경쟁을 넘어 '함께 배우는' 교육적 효과를 노렸습니다.

### 3. 확실한 피드백과 오답 노트 (1인 모드)
- **명확한 오답 팝업**: 1인 모드에서 틀린 답을 선택하면 화면 중앙에 즉시 팝업이 나타나 '내가 선택한 오답', '정답', '상세 해설'을 직관적으로 보여줍니다.
- **만점 축하 이펙트**: 문제를 하나도 틀리지 않고 모두 맞히면 화면 전체에 화려한 축하 애니메이션 팝업 제공.
- **오답 노트 제공**: 게임 종료 후 틀린 문제들만 모아서 해설과 함께 다시 읽어볼 수 있는 복습 기능 제공.
